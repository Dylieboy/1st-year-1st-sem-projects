# Math Tools Library

A reusable C# class library that provides common mathematical operations and utility functions. This project demonstrates how to create custom methods, organize reusable code into a static class, and use namespaces to build modular applications.

---

## 📖 Overview

Instead of placing all calculations inside the `Main()` method, this project separates the mathematical logic into a reusable class library called **MathTools**. A separate console application demonstrates how each method can be used.

This project was created to practice:

- Creating custom methods
- Returning values from methods
- Using parameters
- Static classes
- Namespaces
- Reusable code
- Modular programming

---

## ✨ Features

The library currently includes:

### Basic Arithmetic

- Addition
- Subtraction
- Multiplication
- Division

### Mathematical Functions

- Calculate Hypotenuse (Pythagoras)
- Power
- Square Root
- Factorial
- Average Calculator

### Number Utilities

- Decimal to Binary Conversion
- Even Number Checker
- Odd Number Checker

### Temperature Conversion

- Fahrenheit → Celsius
- Celsius → Fahrenheit

---

## 🛠 Methods Included

| Method | Description |
|---------|-------------|
| `Add()` | Adds two numbers |
| `Subtract()` | Subtracts one number from another |
| `Multiply()` | Multiplies two numbers |
| `Divide()` | Divides two numbers |
| `CalculateHypotenuse()` | Uses Pythagoras' theorem |
| `ToBinary()` | Converts a decimal number to binary |
| `ToCelsius()` | Converts Fahrenheit to Celsius |
| `ToFahrenheit()` | Converts Celsius to Fahrenheit |
| `Power()` | Raises a number to a power |
| `SquareRoot()` | Calculates the square root |
| `Average()` | Calculates the average of numbers |
| `Factorial()` | Calculates the factorial of a number |
| `IsEven()` | Checks if a number is even |
| `IsOdd()` | Checks if a number is odd |
